# tukangAbsen
When engineers doesn't want to fill out attendance manually, tukangAbsen is the answer! (Only For Workforce App)

## How To Use

### Checkin
```
$ tukangAbsen in
```

### Checkout
```
$ tukangAbsen out
```

## Output
```
$ tukangAbsen in
TukangAbsen v0.1

Processing...
Done
Opening Screenshot
```
