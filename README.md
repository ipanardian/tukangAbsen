# tukangAbsen
When engineers don't want to fill out attendance manually...

## How To Use

### Checkin
```
$ tukangAbsen in
```

### Checkout
```
$ tukangAbsen out
```

## Output
```
$ tukangAbsen in
TukangAbsen v0.1

Processing...
Done
Opening Screenshot
```
